FCE Ultra X (fceux) v2.6.6

Лучший эмулятор NES / Dendy и Famicom Disk System с открытым исходным кодом. Поддержка сохранений, джойстиков, эмуляции различных контроллеров, архивов (т.ч. 7z), запись movie и прочее... А также обладает великолепной совместимостью. Портирован под бесчисленное кол-во систем (старое название FCE Ultra). Имеет встроенные инструменты для создания tool-assisted superplay. Qt/SDL-версия имеет более широкий функционал. 

Перевод на русский - Kirlia Team.

Сайт эмулятора - https://fceux.com/web/home.html

Скачено с сайта - https://www.emu-land.net/
